# Calculator-using-HTML-CSS-and-Javascript
## Description: 
A simple calculator built with HTML, CSS, and JavaScript to perform basic arithmetic operations.

## Features: 
Performs basic Addition, Substraction, Multiplication, Division and Modulus operation.
Clear button to reset calculation.

## Technologies used: 
Html, css and Javascript.
